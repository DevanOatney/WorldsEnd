FacingDirection,Duration,ShouldMove,OneShot

EX:
0,1,True,False







Facing Direction:
The direction in which the npc should face;
0-Down, 1-Left, 2-Right, 3-Up

Duration:
The amount of time the NPC should do this action for. (can be decimal)

Should Move:
Flag for if the NPC should move in the direction that they'll now be facing.  (So far there is no run option/speed)

OneShot:
Should the NPC only do this action once?
